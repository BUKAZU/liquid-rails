# Liquid::Rails

Liquid support in Rails. It allows you to render .liquid templates with layout and partial.

## Installation

Add this line to your application's Gemfile:

    gem 'liquid-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install liquid-rails
